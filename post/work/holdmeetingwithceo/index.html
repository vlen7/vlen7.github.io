开6个人的会，其中CEO身价800亿

2021-10-10 次阅读

文章目录

想了想还是用这个标题，毕竟”如何和领导开会“这种题目让人一下子没了兴趣。

照惯例先上链接， 讲的是up主主持6个人的小会议，参会的人都是大佬级别，怎么hold住场面。

为什么要开这个会

  1. 集中做哪一部分
  2. 长期战略
  3. 收购还是自己做
  4. 要资源

注意点

  1. 回答问题简单直接，三到四句话，拖拖拉拉抓不住大佬的注意力
  2. 不要考虑让大家平均发言，CEO没这个时间
  3. 抓住大佬的注意力很关键，回答问题时提出让人能发散思考的观点论点，通过互动抓住注意力
  4. 不要慌乱，要镇得住，不懂的话可以说回去看等等

感想

在这个方面我自己其实也算是有点经验，不算是和大佬开会，只是一些小型的会议，可能有上级和上级的上级。

  1. 首先是不要慌，既然这事轮到你了说明你就是最佳选择，没必要显得慌慌张张。（顺便提一嘴，我之前有一次面试也是因为第一次，然后又是总监面我，显得很是慌张，最后乱了阵脚，这个其实是最忌讳的，当然最后也是被刷掉了）
  2. 其次就是要站在领导的角度去想，领导想听到的是什么，要怎么把达到自己的目的又能让领导听到他想听的东西（这里想听的东西不是指阿谀奉承，而是类似真实情况，面临的挑战，需要的帮助等等）
  3. 然后就是准备充分，最好能有稿子，清单这些帮助你理清思路的东西。很多时候我其实不太愿意去做这些，但是真的到了你主持会议的时候，很容易就忘记说了某个或某些重要的点，这些失误可能不致命，但是会让这次会议的效果大打折扣，要么是你的目的诉求没有传达到，要么是领导想听的东西没听到（大佬一般都没有那个精力去挖掘你的成绩，自己说出来最靠谱，不要等着别人问你）。推荐《清单革命》这本书，帮助你认识清单的重要性。
  4. 细枝末节点到即可。真的不要长篇大论说一堆大家不关心的东西，你说了要么是被打断，要么是没人听。抓住大家的注意力是关键，
  5. 有不会的问题，如果是关键问题，大胆承认并改正，不要想着糊弄。但如果是细枝末节，随便忽悠过去就行，或者说会后讨论等。这里提一句之前同事教我的，有些奇奇怪怪的问题不好回答，你就随便说一些看似相关实则不对题的答案。既然问题不好好问，我们何必好好回答呢~

能想到的大概就这些吧，个人感觉职业路还很长啊，自己要学的东西真的太多了。想成为大佬没那么简单呀。CEO一天几千封邮件，几十个会议，还要提出建设性意见，指挥大家走向正确的道路。我啥时候能有这本事呢？
